# Conexão com Banco — Funcionário e Dependente

CRUD de funcionários e dependentes desenvolvido em Java com padrão DAO e banco MySQL.

## Funcionalidades
- Listar, inserir, buscar, atualizar e excluir funcionários e dependentes
- Listagem combinada funcionário + dependente via JOIN
- Conexão com MySQL via JDBC (MySQL Connector/J)

## Como executar
Configure o banco `exercicio` no MySQL local (porta 3306, usuário `root`, sem senha) e execute `AulaDAO.java`.
